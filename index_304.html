COMPX304 Mini-Internet Project

This website hosts some of the debugging resources for the mini-Internet assignment. These tools are similar to those deployed on the real Internet. You can find the assignment itself and other resources on the course Moodle page.

Route Tables (Looking Glass)

You can access the routes that every router on the mini-Internet Access here: looking glass
These are ordered by AS number then router name. For example, the Tier1 AS 1 routes for HAML are here: looking_glass/G1/HAML.txt

Connectivity Matrix

The connectivity matrix actively probes reachability within the mini-Internet. It shows which ASes are reachable from others.
You can access the connectivity matrix here: matrix/matrix.html

Connection List

The list of links connecting ASes, the relationship with the AS on the link, and the addresses to use on that link.
Access here: as_connections.txt

BGP Analyzer

This tool shows if an AS is following the correct policy with its announcements over the IXP.

Note: This tool only detects incorrect announcements, not missing announcements. If you are not sharing any routes over BGP your configuration is incorrect, but this tool won't detect that case. It doesn't guarantee that your configuration is correct.

Access here: analysis.html
